# Management System

A premium Admin Dashboard built with React, Vite, and Glassmorphism CSS.

## Features

- **Authentication**: Secure mock login (email/pass).
- **Dashboard**: Interactive stats and charts.
- **User Management**: Add, Edit, Delete users (CRUD simulation).
- **Responsive Design**: Works on all devices.
- **Glassmorphism Theme**: Modern UI with real-time blur effects.

## Getting Started

1.  navigate to the project folder:

    ```bash
    cd management-system
    ```

2.  Install dependencies (if you haven't already):

    ```bash
    npm install
    ```

3.  Run the development server:
    ```bash
    npm run dev
    ```
